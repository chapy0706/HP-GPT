# PCフッター（ホバー展開）撤去 / お問い合わせへ営業時間・地図を移設

## 背景
PC版でイントロ完了後のTOP画面下部フッターにカーソルを合わせると、展開エリア（`.footer-info-expand`）が開き、営業時間・地図（＋小カレンダー）が表示される。

このホバー展開（アコーディオン表示）は、見た目の意図が伝わりにくく、表示崩れの温床にもなりやすい。  
代わりに、ユーザーが自然に辿り着く「お問い合わせ」ページに情報を集約したい。

## ゴール
- PCフッターのホバー展開（`.footer-info-expand`）を無くす（＝表示も挙動も撤去）
- 「お問い合わせ」ページ（`#contact`）の **営業カレンダー（大）** の下に、フッターに置いていた
  - 営業時間
  - 地図
  を表示する
- イントロ演出 / 遷移 / 既存セクションの挙動への影響を最小にする
- 依存追加なし（ネットワーク無し環境で `make verify` が通る）

## 非ゴール
- フッター全体の再デザイン
- カレンダーや地図をAPIで取得して独自UIに置き換える（iframeのまま）

---

## 対象範囲
- `/index.html`
  - footer: `.footer-info-expand` を撤去
  - contact: `#contact` のカレンダー直下へ営業時間・地図を移設
- `/style.css`
  - `.footer-info-expand` のアコーディオンCSSを無効化
  - `#contact` 下の営業時間・地図レイアウト（レスポンシブ）を追加
- `/script.js`
  - 原則変更なし（ホバー展開はCSS制御のため）

---

## 仕様
### S1: フッター（PC）
- `.footer-info-row` にホバーしても展開エリアが出ない
- フッターは住所・電話など「コンパクト情報」のみ表示する
- フッター内の案内文は実態に合わせて更新する（営業時間・地図はお問い合わせに掲載）

### S2: お問い合わせ（#contact）
- `#contact` の「営業カレンダー」の直下に「営業時間」「地図」を表示する
- 横幅が広いときは 2カラム、狭いときは 1カラムで崩れない
- 地図は既存の Google Maps iframe を流用して良い

---

## 受け入れ条件（AC）
- AC1: PCでフッターにカーソルを合わせても、営業時間・地図などの展開表示が出ない
- AC2: `#contact` のカレンダーの下に営業時間と地図が表示される
- AC3: イントロ演出 / ページ内遷移 / 他セクションが壊れていない
- AC4: 依存追加なしで `make verify` が通る

---

## 実装タスク
### T1: index.html
- `.footer-info-expand` を削除（または常時非表示化）し、フッター案内文を更新
- `#contact` のカレンダー直下に、営業時間と地図（既存フッター内容）を移設して表示

### T2: style.css
- `.footer-info-expand` のアコーディオン用CSSを無効化（hoverで開かない）
- `#contact` の営業時間・地図用のレイアウトを追加（2カラム→1カラム）

### T3: script.js
- 変更不要（想定）

---

## 手動確認
1. `index.html` を開き、イントロ完了後にTOP画面へ
2. フッターにカーソルを合わせても展開表示が出ないことを確認
3. フッターナビから「お問い合わせ」へ遷移
4. カレンダーの下に営業時間と地図が表示されることを確認
5. 画面幅を狭めても崩れないことを確認

## Claude Code 用プロンプト（コピペ用）
目的: PC版フッターのホバー展開（`.footer-info-expand`）を撤去し、代わりに `#contact` のカレンダー直下へ「営業時間」と「地図」を移設して表示したい。依存追加は禁止（ネットワーク無しで `make verify` を通す運用）。イントロ演出やページ遷移を壊さず、差分最小で実装してほしい。

作業:
1) `/index.html` のフッター内 `.footer-info-expand` を削除（または常時非表示）にし、`.footer-info-hint` を「お問い合わせに掲載」に更新
2) `/index.html` の `#contact` セクションで、カレンダーの下に営業時間と地図を表示（フッターにあった内容を移設）
3) `/style.css` のフッターアコーディオン用CSS（hoverで展開）を無効化
4) `/style.css` に `#contact` 用の「営業時間 + 地図」レイアウトを追加（2カラム→モバイルで1カラム）
5) `make verify` を通す

注意:
- 既存の id / class を削除・改名しない
- ついでリファクタ禁止
- 影響範囲はフッターと `#contact` に限定
